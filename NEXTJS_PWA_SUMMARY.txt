QUICK START (5 MINUTES)
═════════════════════════════════════════════════════════════════════════════

1. Install Dependencies
   cd artifacts/barangay-portal-nextjs
   npm install

2. Create Environment File
   Create .env.local with:
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
   OPENAI_API_KEY=your_key

3. Start Development Server
   npm run dev
   
4. Visit Application
   http://localhost:3000

5. Test PWA (in Chrome)
   F12 → Application → Manifest
   Look for "Install app" prompt

MIGRATION PATH FROM VITE PROJECT
═════════════════════════════════════════════════════════════════════════════

1. Copy Components
   cp -r old-project/src/components/* src/components/
   cp -r old-project/src/lib/* src/lib/

2. Update Imports (relative to absolute)
   OLD: import { Button } from "../components/ui/button"
   NEW: import { Button } from "@/components/ui/button"

3. Move Pages
   OLD: src/pages/dashboard.tsx
   NEW: src/app/dashboard/page.tsx

4. Move API Routes
   OLD: src/pages/api/documents.ts
   NEW: src/app/api/documents/route.ts

5. Update API Response
   OLD: res.json({ data })
   NEW: Response.json({ data })

6. Copy Styles
   cp old-project/src/index.css src/app/globals.css
